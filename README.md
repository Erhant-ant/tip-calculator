# Tip Calculator

A simple Python tip calculator application.

## Usage

```bash
python main.py
```

## Features
- Enter the total meal price
- Choose tip percentage (10%, 12%, or 15%)
- Split the bill among multiple people
- Calculate how much each person pays

## Example Output

```
Welcome to the Tip calculator

Total meal price: 100
How much tip would you like to give? (10, 12, or 15): 15
How many people to split the bill?: 4

Total cost is 115.0$, 15% you tipped and each person will pay 28.75
```

## Requirements

- Python 3.x

## How It Works

1. Enter the total meal price
2. Select the tip percentage you want to give
3. Enter how many people to split the bill
4. Get the amount each person needs to pay
